1. Responsable del Tratamiento

  • Correo Electrónico: Maderofernanfo@gmail.com

2. Finalidad del Tratamiento de Datos

Los datos personales recogidos serán utilizados para:

  • Atender consultas y solicitudes enviadas a través de los formularios de contacto.
  • Enviar información comercial y promociones (si el usuario lo ha autorizado).
  • Gestionar compras y pedidos.
  • Analizar el tráfico web mediante cookies.

3. Legitimación para el Tratamiento de Datos

El tratamiento de datos personales se realiza en base a:

  • El consentimiento del usuario (marcando las casillas correspondientes en los formularios).
  • La ejecución de un contrato en caso de compra de productos.
  • El cumplimiento de obligaciones legales.

4. Derechos del Usuario

Los usuarios pueden ejercer los siguientes derechos sobre sus datos personales:

  • Acceder a sus datos personales.
  • Solicitar la rectificación de datos incorrectos.
  • Pedir la supresión de sus datos cuando ya no sean necesarios.
  • Oponerse al tratamiento de sus datos para fines comerciales.
  • Solicitar la portabilidad de sus datos.

Para ejercer estos derechos, los usuarios pueden enviar un correo a Maderofernanfo@gmail.com con una copia de su documento de identidad.

5. Seguridad y Conservación de los Datos

La Tienda Latina adopta medidas de seguridad para proteger los datos personales. Los datos se conservarán mientras sean necesarios para cumplir con la finalidad para la que fueron recogidos o para cumplir obligaciones legales.

6. Modificaciones en la Política de Privacidad

Nos reservamos el derecho de modificar esta Política de Privacidad para adaptarla a novedades legislativas o cambios en nuestras actividades. Se recomienda revisarla periódicamente.
